៣០. មិថុនា. ២០២៣

ពេលវេលារាប់ថយក្រោយនៃថ្ងៃរៀបអាពាហ៍ពិពាហ៍

កម្មវិធីលំអិត

កម្មវិធីថ្ងៃទី១

ថ្ងៃព្រហស្បតិ៍ ទី២៩ ខែមិថុនា ឆ្នាំ២០២៣

  • ម៉ោង ០១:៣០នាទីរសៀល : ជួបជុំបងប្អូន និងភ្ញៀវកិត្តិយស

  • ម៉ោង ០២:០០នាទីរសៀល : ពិធីសែនក្រុងពាលី

  • ម៉ោង ០៣:០០នាទីរសៀល : ពិធីសូត្រមន្តចំរើនព្រះបរិត្ត

  • ម៉ោង ០៤:០០នាទីរសៀល : ពិធីជាវខាន់ស្លា និងជុំកោលា

  • ម៉ោង ០៥:០០នាទីល្ងាច : អញ្ជើញភ្ញៀវកិត្តិយសពិសាអាហារពេលល្ងាច

កម្មវិធីថ្ងៃទី២

ថ្ងៃសុក្រ ទី៣០ ខែមិថុនា ឆ្នាំ២០២៣

  • ម៉ោង ០៦:៣០ នាទីព្រឹក : ជួបជុំបងប្អូន និងភ្ញៀវកិត្តិយសរៀបចំហែជំនូន

  • ម៉ោង ០៧:០០ នាទីព្រឹក : ពិធីហែជំនូន នៅគេហដ្ឋានខាងស្រី

  • ម៉ោង ០៧:៣០ នាទីព្រឹក : មេបា និយាយជើងការ និងរៀបចំផ្លៃឈើ

  • ម៉ោង ០៨:០០ នាទីព្រឹក : ពិធីបំពាក់ចិញ្ជៀន

  • ម៉ោង ០៨:៣០ នាទីព្រឹក : ពិធីសែនព្រេនកុងម៉ា

  • ម៉ោង ០៩:០០ នាទីព្រឹក : ពិធីកាត់សក់បង្កក់សិរី

  • ម៉ោង ១០:០០ នាទីព្រឹក : ពិធីបង្វិលពពិល សែនចងដៃ សំពះផ្ទឹម និងបាចផ្កាស្លា

  • ម៉ោង ១១:៣០ ថ្ងៃត្រង់ : អញ្ជើញភ្ញៀវកិត្តិយសពិសាអាហារថ្ងៃត្រង់

  • ម៉ោង ០៤:៣០ នាទីល្ងាច : អញ្ជើញភ្ញៀវកិត្តិយសពិសាភោជនាហារ
